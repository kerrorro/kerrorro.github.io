Swift Subset Interpreter

Jython // Python Lex-Yacc

For the final project of the Elements of Programming Languages course taught at The University of Texas at Austin, Fahran Kamili and I decided to create a Python intepreter for a subset of the Swift programming language. This included variable assignments, arithmetic expressions, print statements, and switch-case statements. In order to do this, tokens are defined at the base level as groups of characters that can be recognized by the interpreter. The tokens are then strung together into various grammar rules that define an acceptable Swift statement, and the expected output is done so in Python.
