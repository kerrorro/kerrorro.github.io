HBC Cookbook

Python // BeautifulSoup4 // PyMySql

HBC Cookbook is a command line interface recipe book specificially for saving recipes obtained from www.foodnetwork.com. Home cooks enter a url, and the HBC Cookbook will parse through the recipe to find relevant information. They can then search for their recipes by chef, category, ingredients, max cook time, or difficulty.

HBC Cookbook was created as part of the Elements of Databases course taught at The Univeristy of Texas at Austin by me, Happy Situ, and Brent Tarnopol. Our roles for the project were well-defined. We first collaborated on creating a database ERD schema from which Happy wrote SQL statements for table creation. In order to populate the database as part of the demo, I wrote a crawler and recipe parser using BeautifulSoup4, a Python library that extracts HTML elements. I then used PyMySql to insert the extracted information into our tables. Brent was responsible for the command line interface and user-prompting.
